# Backend Docs

This directory stores deep technical documentation for the backend. Documents in this folder should:

1. Focus on implementation depth, trade-offs, and operational guidance.
2. Include citations to external references (official docs, specs, or code links).
3. Avoid changelog-style entries; use task JSON files and VCS history for that.
